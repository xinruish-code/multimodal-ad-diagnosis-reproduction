# Results

This folder contains reproduced results, tables, figures, and experiment logs.

## Planned Structure

- `tables/`: summary tables for classification results
- `figures/`: plots, ROC curves, confusion matrices, and visualizations
- `logs/`: training and evaluation logs

## Metrics

The main evaluation metrics include:

- Accuracy
- Sensitivity
- Specificity
- AUC
- F1-score
- Precision
- Recall
- Balanced Accuracy
