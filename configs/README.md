# Configs

This folder contains experiment configuration files for different reproduced models.

## Planned Config Files

- `mdl_net.yaml`: configuration for MDL-Net reproduction
- `mmrn.yaml`: configuration for MMRN reproduction
- `la_gmf.yaml`: configuration for LA-GMF reproduction
- `dhanet.yaml`: configuration for DHANet reproduction
- `aagn.yaml`: configuration for AAGN reproduction
- `cgan_ad.yaml`: configuration for CGAN-AD reproduction
- `unicross.yaml`: configuration for UniCross reproduction

## Notes

Each config file will include settings such as:

- model name
- dataset path
- modality type
- classification task
- batch size
- learning rate
- number of epochs
- cross-validation setting
- output directory
