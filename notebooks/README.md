# Notebooks

This folder contains notebooks for data checking, visualization, and result analysis.

## Planned Notebooks

- `data_check.ipynb`: check data paths, labels, modalities, and missing values
- `result_analysis.ipynb`: analyze reproduced classification results
- `visualization.ipynb`: visualize result tables, ROC curves, and confusion matrices

## Notes

Notebooks are mainly used for exploration and analysis. Formal training and evaluation scripts should be organized under the `src/` folder.
